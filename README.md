# بوابة شفائي (Shifae Portal)

## وصف المشروع
نظام إلكتروني متكامل لإدارة الاستقبال وإدخال المرضى، يهدف إلى تنظيم العمليات الإدارية في المراكز الطبية. يتضمن النظام خوارزمية ذكية للتحقق اللحظي من توافر الحجرات ومنع التعارض الزمني في المواعيد، مما يضمن كفاءة توزيع المرضى على الأطباء والمرافق المتاحة.

## المتطلبات التقنية (Tech Stack)
* الإطار البرمجي: Laravel
* قاعدة البيانات: MySQL
* الواجهات: Blade Templates, CSS, JavaScript

## تعليمات التجهيز (Build Instructions)
لتجهيز المشروع في بيئة العمل المحلية، يرجى تنفيذ الأوامر التالية بالترتيب:

1. تنصيب المكتبات البرمجية:
   composer install
   npm install && npm run build

2. إعداد ملف الإعدادات:
   * قم بإنشاء ملف `.env` بناءً على `.env.example`.
   * قم بإنشاء قاعدة بيانات MySQL محلية وربطها في ملف `.env`.

3. تجهيز قاعدة البيانات:
   php artisan key:generate
   php artisan migrate

## طريقة التشغيل (Run Instructions)
بعد اكتمال الإعداد، يمكن تشغيل النظام كالتالي:

1. تنفيذ أمر تشغيل السيرفر:
   php artisan serve

2. الدخول للنظام عبر المتصفح:
   http://localhost:8000
