# Shifae Portal - نظام الاستقبال الإلكتروني (بوابة شفائي) 🏥

نظام إلكتروني متكامل لإدارة عمليات استقبال المرضى وتنسيق المواعيد، مصمم لتبسيط الإجراءات الإدارية وضمان عدم التعارض الزمني في المركز الطبي.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Framework:** Laravel 10+
* **Database:** MySQL (Via WAMP/XAMPP)
* **Frontend:** Blade, CSS, JS
* **Design Pattern:** MVC & SOLID Principles

## 📋 المتطلبات الأساسية (Prerequisites)
قبل البدء، تأكد من تنصيب البرامج التالية:
* PHP >= 8.1
* Composer
* Node.js & NPM
* WAMP Server (لتشغيل MySQL)
